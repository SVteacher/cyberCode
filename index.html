How to Use

  • Grading Output is for student feedback.
  • Email Generator is for recurring, consistent student communication.
  • Copy first, then make only the smallest custom edit needed.
  • Use Save Override only when you want your wording saved on this device.

Communication Flow

Consistent communication

  • Welcome email at course start.
  • Early check-in after add/drop settles.
  • 25%, 50%, and 75% progress reminders.
  • Next nine weeks / next part loaded notice before rollover.
  • Semester or end-of-year closeout based on course structure.

SV Portal Final / Weekly Grade Setup

  • Prior to the start of each nine-week grading period, use the following to set up a weekly grade reminder.
  • At the end of each grading period, you will add the final grade placeholder, enter grades, then delete the weekly grade update.

Weekly Overall Grade Update

Assignment Name: Weekly Overall Grade Update

Assignment Detail: This grade will be updated weekly to serve as the student's current overall grade for the course. Individual assignment grades are located on CL / Microsoft Teams.

Final Overall Grade

Assignment Name: Final Overall Grade

Assignment Detail: Student's final grade for this nine-week grading period. Individual assignment scores can be found in student's profile / gradebook in Microsoft Teams / CL.
